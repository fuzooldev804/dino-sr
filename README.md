🦖 dino-sr

Retro Speed Roaster — Dino Edition

dino-sr is a lightweight, retro-themed internet speed test that roasts your connection in real-time. Built with a handheld console aesthetic, it features a running Dino whose pace matches your bandwidth, delivering bilingual (English/Urdu) roasts based on how "potato" your internet is.

🚀 Live Demo

Check it out here: https://fuzooldev804.github.io/dino-sr/

✨ Features

Accurate Sampling: Uses chunk-based download streams for real-time Mbps calculation.

GameBoy Aesthetic: Pure CSS console hardware design with a pixel-perfect canvas runner.

Bilingual Roasts: Humorous feedback in English and Urdu tailored to your speed bracket.

Instant Sharing: Built-in "Share on X" functionality with pre-filled roast data.

Fully Responsive: Optimized for both desktop and mobile "handheld" experiences.

🛠️ Installation & Deployment

Since this is a single-file Vanilla JS application, deployment is instant:

Create a new repo named dino-sr.

Upload index.html.

Enable GitHub Pages in Settings.
